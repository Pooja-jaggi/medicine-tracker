# MediTrack - Medicine Expiry Tracker

## About
MediTrack is a web application that helps users track medicine expiry dates and manage their medication inventory.

## Features
- User authentication (register/login)
- Add, edit, delete medicines
- Track expiry dates
- Low stock alerts
- Automated email reminders via Flask-Mail
- Dashboard with medicine overview

## Tech Stack
- **Backend:** Python, Flask
- **Database:** SQLite
- **Frontend:** HTML, CSS, JavaScript, Jinja2 templates
- **Email:** Flask-Mail
- **Deployment:** Render
